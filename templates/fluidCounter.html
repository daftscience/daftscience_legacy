Keyboard layout

Calculations

Counting Cells

Start counting the cells on the first side of the Hemocytometer as you normally would. The "W" key is used for Leukocytes, and the "R" key is used for Erythrocytes.

Area

This is where you set the counting area per side. The calculation assumes that you counted the same area on each side of the hemocytometer.

Dilution Factor

In some cases it might be useful to use a different dilution factor for each cell type. If you're diluting the sample, and counting both types with that dilution, make sure to enter it into both fields.

Crystal Violet

A 9:10 dilution with crystal violet makes differentiating leukocytes from erythrocytes much easier when performing counts. If it was used simply check the box. The calculation assumes it was used for both cell types.

Switching sides

When you've finished counting the first side hit the "E" key to toggle to the next side, or simply press the button.

What is this?

This is an application to help count cells on a hemocytometer. After you've finished counting your cells, the calculated value will be displayed.

DISCLAIMER:

It is good practice to check that the calculations provided by this application are correct. If the area and dilution factor shown in the calculations are correct then the calculated value should be correct as well. However, solar flairs or internet goblins may mess with the numbers, so please double check.
