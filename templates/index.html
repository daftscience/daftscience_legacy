The RostockMax is a delta robot 3d printer designed by SeeMeCNC. The three arm design allows this printer to have a combination of speed, build volume, precision and accuracy that (at the time it was made) was not easy to replicate with a standard cartisian printer. However, the design also makes this printer more difficult to build and calibrate.

The Prusa i3 (iteration 3), designed by Josef Prusa, is the third 3d printer in the Prusa line of printers. It's an H-frame cartisian printer with a simple design that is both easy to build and calibrate. However, the build volume is much smaller, and single plate design makes it unstable at higher speeds (box framed versions compensate for this.)

The Simpson printer, designed by Nicolas Seward, is an entirely new beast. This printer is made almost entirely out of 3d printed parts. The drive mechanism is made from fishing line instead of belts (which can get pricey, especially on the rostock.) This design is still in its infancy and there isn't any software that has supports it, meaning the gcode (a set of instructions the printer uses to print an object) needs to be run through a pre-processor script before sending it to the printer.
