Daftscience Diffcounter

The goal was to make a web based differential counter for students of biomedical science that was not only easy to use, but that would emulate software used in a clinical setting. Recently, the focus has been on modularization. The templates were built using Flask and some json files. By modifying the json files it is possible to change not only the layout of the pages, but the keymapping as well. While the original goals have been met, there are still a few more features that are in the works.

RostockMax

Prusa i3

The RostockMax is a delta robot 3d printer designed by SeeMeCNC. The three arm design allows this printer to have a combination of speed, build volume, precision and accuracy that (at the time it was made) was not easy to replicate with a standard cartesian printer. However, the design also makes this printer more difficult to build and calibrate.

The Prusa i3 (iteration 3), designed by Josef Prusa, is the third 3d printer in the Prusa line of printers. It's an H-frame cartisian printer with a simple design that is both easy to build and calibrate. However, the build volume is much smaller, and single plate design makes it unstable at higher speeds (box framed versions compensate for this.)
